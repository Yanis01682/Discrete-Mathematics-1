# Discrete Mathematics (1)

Course-related files for Tsinghua University **Discrete Mathematics (1)**, Fall 2024.

This repository is a cleaned archive of my course materials, homework-related files, exam review files, and small code snippets. Files with obvious personal information in their filenames were intentionally omitted.

## Contents

- `lectures/`: lecture slides and syllabus materials
- `homework/`: homework files
- `exams/`: review and exam-related files
- `problem-writing/`: problem-writing assignment materials
- `code/`: small course-related code snippets
